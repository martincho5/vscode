Please make the following changes on the morning of the first day before we begin:


* In your testing VM, not on your host laptop, create this folder:  C:\SANS

* Copy this entire CD into C:\SANS inside your VM. 

* In your VM, right-click the blue PowerShell shortcut in the taskbar and Run As Administrator.

* In PowerShell, type the following commands:


    Set-ExecutionPolicy Bypass -Force
    cd C:\SANS
    .\SEC505-Setup-Script.ps1


* Follow the instructions from the setup script.  Don't forget that you will need to run the script *twice*, as instructed by the script.
